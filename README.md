Content Management (Spring Boot + MySQL)

This repository contains both the Admin Facing CMS application as well as the Spring Boot API that manages it,  

Features:

    Create blog posts (POST)
    Fetch all blog posts (GET)
    Filter through set genres(GET by Filter)
    Update blog posts (PUT)
    Delete blog posts (DELETE)
    JSON-based REST API
    Error handling + response messages

Tech Stack:

    Java 17
    Spring Boot
        Spring Web
        Spring Data JPA|JDBC
        Maven
    REST API
    Android Studio
        retrofit
